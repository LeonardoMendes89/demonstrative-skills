media de proporção em dispositivos:

  estes valores estudados
  são médias aproximadas,
  para que eu tenha uma
  base em valores na resolução 
  de interfaces dos dispositivos
  com o intuito de aplicar técnicas
  de responsividade mais rápido,
  mais produtivo e mais especializado.*


 (   ++  ) a mais ou acima
 (   --  ) a menos ou para baixo
 (   -   ) entre
 (   ~   ) aproximadamente
  
  desktop/tv:
	  1024px~ ++

  tablet:
	  768px~ - 1024px~

  celulares:
	  240px~ - 768px~

  wearables:
  	  218px~ - 281px~
